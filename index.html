Bitcoin Price Checker

Quick and simple program that checks the bitcoin price in a small, simple window.

Download as .zip Download as .tar.gz View on GitHub

Project Overview

I have created a simple app using Python 2.7 and wxPython that checks the price of the Bitcoin price and changes the background color to show if it went up or down from the last trade. You can also make a chart with a drop-down menu of some time frames. The app was made for windows, but the app should work on other platforms with minor tweaks to the frame and positioning of widgets. You can select how often you want it to update and in what currency you want it in. The BTC quotes are from coindesk.com/price

The Code

I tried to comment as much as possible so it is easy to pick up if you want to add your own features to it. I have two code files for the project, one for the main frame and one for the settings panel.

How do I download it?

All you need to do is either clone the repository or download it as a zip and install Python 2.7 with the module wxPython if you computer doesn't already have it. I will have the packaged .exe version as soon as possible and I will post the link as soon as I put it up.
